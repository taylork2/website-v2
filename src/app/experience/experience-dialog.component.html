Computer Science Capstone @RDE Systems

FEB 2018-MAY 2018

About

RDE Systems is a technology consultancy company that serves the public health sector. As part of a small team, I designed and developed a messaging system using Angular to facilitate communication between AIDS patients and their caseworkers. I also created a Coldfusion API and integrated this messaging system into a larger webapp which ensured patients living with AIDS and HIV have access to medication.

Task

  • Within 3 months, create a secure two-way messaging system
  • Be able to send, receive and view mail
  • Easily search users
  • Create different roles for case workers and patients which have different abilities
